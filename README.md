# BMIndex

A simple BMI Calculator.
UI desiged by [Ruben Vaalt](https://dribbble.com/rvaalt) 

## Getting Started

### Installation
1. Clone repository
   
    ```sh
    git clone https://github.com/Deva0813/BMIndex.git
    ```
    
2. Install Packages

    ```sh
    flutter pub get
    ```
   
3. Run App for Developement

    ```sh
    flutter run
    ```
